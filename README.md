# NBA App

iOS and Android NBA app created with React Native.

## Tech Stack

- React Native
- Gesture Handler
- Reanimated 3
- TypeScript
- TailwindCSS
- React Navigation
  - React Screens
  - React Safe Area Context
  - React Native Blur
- React Native Paper
- React Vector Icons
